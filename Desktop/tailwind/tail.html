Tailwindcss

Tailwind CSS is a highly customizable, low-level CSS framework that gives you all of the building blocks you need to build bespoke designs without any annoying .

They come with all sorts of predesigned components like buttons, cards, and alerts that might help you move quickly at first, but cause more pain than they cure when it comes time to make your site stand out with a custom design.

Instalation

For most projects (and to take advantage of Tailwind's customization features), want to install Tailwind via npm.

Before using the CDN build, please note that many of the features that make Tailwind CSS great are not available without incorporating Tailwind into your build process.

# Using npm

npm install tailwindcss

# Using Yarn

yarn add tailwindcss

Tailwindcss Icons

For the first time ever, Tailwindcss has its own open source SVG icon library, designed to work best with our components and documentation.

Tailwindcss Icons are designed to work best with Tailwindcss components, but they’ll work in any project. They’re SVGs, so they scale quickly and easily, can be implemented in several ways, and can be styled with CSS.

TailwindcssCDN

When you only need to include Tailwindcss’s compiled CSS or JS, you can use TailwindcssCDN.
